# P5play initial project 

Initial project with p5play as dependancy

## Getting Started


### Prerequisites

You'll need npm 

### Installing

Clone the repo

```
git clone git@gitlab.com:Spymannn/phaser_base.git
```
Enter to the directory 

```
cd p5play_clean
```

Install dependancies

```
npm install
```

Run the application

```
npm start
```
